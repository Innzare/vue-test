# База данных mysql
База данных состоит из одной таблицы workers со следующими полями
``` 
 - id FKEY, INT NOT NULL
 - name VARCHAR NULL
 - department VARCHAR NULL
 - expierence VARCHAR NULL
 - phone VARCHAR NULL
```
Слепок БД лежит в `stuff/vue-test.sql`
